## Contrast Checking Quick Reference
| Ratio | WCAG Level | Use For |
|-------|-----------|---------|
| 7:1+ | AAA | Body text, critical info |
| 4.5:1+ | AA | Standard text (minimum) |
| 3:1+ | AA Large | Headings 18px+, UI components |
| < 3:1 | FAIL | Never use for text or interactive elements |

## Dark Mode Translation Rules
| Light Token | Dark Equivalent | Rule |
|------------|----------------|------|
| White bg | gray-900 (not pure black) | Reduce brightness, not invert |
| gray-50 surface | gray-800 surface | One step lighter than bg |
| Primary-600 | Primary-400 | Lighten 2 steps for visibility |
| Text gray-900 | Text gray-100 | High contrast maintained |
| Border gray-200 | Border gray-700 | Subtle but visible |
| Shadow black/10% | Shadow black/40% | Stronger for depth on dark |
