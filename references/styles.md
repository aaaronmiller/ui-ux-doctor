## Migration Paths (Style Evolution)
| From | To | Rationale | Key Changes |
|------|----|-----------|-------------|
| Flat 2.0 | Bento Grid | Adds depth without gradients | Variable card sizes, subtle shadows |
| Bootstrap Default | Tailwind Custom | Eliminates sameness | Remove component library look, custom tokens |
| Material Design | Custom System | Escape Google association | Custom elevation, unique motion curves |
| Skeuomorphism | Glassmorphism | Modern depth metaphor | Translucency over texture |
| Generic SaaS | Swiss Grid | Add sophistication | Strict grid, limited palette, better type |
| Dark Bootstrap | Luxury Dark | Elevate perception | Gold/cream accents, serif headings, spacing |
